ATOM Autopilot

The ATOM flight control system enables fully autonomous flights for UAS

ATOM is an autopilot system that enables autonomous flights for fixed wing UAVs. ATOM is a fully featured RoHS compliant autopilot that incorporates several innovative features such as an intelligent landing system, health monitoring and onboard data recording in an incredibly small form factor.

SALIENT FEATURES

    • Small size (4 cm x 4 cm) and lightweight (13 grams)
    • Autonomous flight control from takeoff to landing
    • Dynamic in-flight updates of the flight plan and real-time tuning of autopilot parameters
    • Onboard data logging on microSD card at a high rate
    • Built-in tests for increased reliability and safety
    • Health monitoring with failsafe features
    • Hardware-in-the-loop simulation for testing and training
    • Configurable to support a variety of UAV airframes

ATOM interfaces with the operator through Asteria’s proprietary Mission Control Software (MCS).  The MCS software is a cross-platform graphical user interface application that runs on Windows and Linux operating systems. The MCS receives and displays telemetry data from the ATOM autopilot and allows the user to command, control and configure the autopilot.

SPECIFICATION

PHYSICAL

Size

4 cm x 4 cm x 1.5 cm

Weight

13 grams excluding external sensors and without enclosure. 30 grams with enclosure.

Power

8 to 42 VDC input. The autopilot consumes less than 50mA at 12 V excluding external sensors, radio, servos

Operating Temperature

-10 to +50 °C

Sensors
  • 3 axis accelerometer
  • 3 axis gyroscope
  • 3 axis magnetometer
  • GPS
  • Airspeed sensor
  • Barometric pressure sensor
  • Battery voltage and current sensor

Storage

4 GB removable microSD card for onboard data log storage

 

 

SOFTWARE

Flight Control Modes

Besides Manual and Stabilize control, the following normal Auto flight control modes are supported:

  • Takeoff
  • Heading Hold
  • Fly-by-wire
  • Navigation
  • Loiter
  • Return to Home
  • Intelligent Landing
  • Autothrottle in all modes for airspeed/throttle control

Navigation

A flight plan with up to 250 navigation points is supported by the autopilot. Flight plan can be updated in real time.

 

 

USER INTERFACE (MCS)

Salient Features
  • Intuitive graphical user interface that enables full command and control of the autopilot
  • Display of autopilot telemetry and status information in a cockpit format
  • Point-and-click interface with a moving map to enable the user to program or update the autopilot flight plan with ease
  • Health monitoring and display of messages, alerts and warnings
  • Capability to configure the autopilot and tune parameters in real time for flight control
  • Display of real-time telemetry in graphs for in-flight analysis
  • Capability to record flight data on the ground at 10 Hz and play back for post flight analysis
  • Hardware-in-the-Loop flight simulation for training

                        MZP Hqrs. chuan dt.18.6.2007 khan OB meeting an nei a, an meeting chuan Mizoram-Assam border Vairengte bulah ZO RUN sa turin resolution an pass a. A tuk dt.19.6.2007@7:00pm ah bus pathum hmangin MZP member te ZO RUN sa turin an chhukthla ta ani.

 

                        Dt.20.6.2007 khan ZO RUN sak an tan a, Assam Police te nen hun rei vaklo chhung chu an buai a, mahse ZO RUN chu an sak chhunzawm ta tho ani. Chutih laiin he ZO RUN hi sak zawh fel anih hnu lawkah Mizoram sorkar thupek angin Kolasib Add.SP hovin hemi ni tlaiah hian Mizoram Police ten an thiat leh nghal ani.

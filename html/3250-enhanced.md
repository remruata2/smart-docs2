A400

The A400 is a mini VTOL UAS for surveillance and security applications

[Photo: A400 VTOL UAV]

The A400 mini UAS combines best-in-class range &amp; endurance with advanced payload capability forsurveillance and security applications. The system provides real-time high resolution video for day or night operations with advanced features for monitoring and analysis. Its compact size and verticaltakeoff/landing capability make it ideal to operate in a constrained environment.

SALIENT FEATURES

- **Autonomous**– Automatic takeoff, navigation and landing
- **Ease of use**–Asteria’s intuitive software for command and control
- **Network**enabled ground system architecture
- **Secure**– Encrypted digital data link for secure transmission of data and video
- **Safety**– Failsafemodes for loss of communication signal, GPS signal and low battery
- **Construction**– Hybrid composite/metal framework
- **Power**– Electric with rechargeable batteries
- **Packaging**– Complete system in two rugged, waterproof backpacks
- **Made in India**– Designed, Developed and Made in India
